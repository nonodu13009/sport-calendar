## 1) Introduction et Principes Fondamentaux

### 1.1 Spécificité des Sports de Combat
- **Profil énergétique** : Efforts intermittents intenses (alactiques très courts), phases isométriques/tirées plus longues (lactique), **forte contribution aérobie** pour tenir les rounds/journées
- **Qualités requises** : Tirages/poignées, hanches/chaîne postérieure, gainage anti‑rotation, **puissance alactique (6–10 s)**, capacité à répéter l'effort
- **Priorités** : 1) Sommeil & santé 2) Technique/tatami 3) Force/puissance 4) Métab. spécifique

### 1.2 Principes de Périodisation
- **Macrocycle** : Planification annuelle (10-12 mois)
- **Mésocycle** : Périodes de 3-6 semaines
- **Microcycle** : Semaine d'entraînement
- **Surcompensation** : Phase de charge → Fatigue → Récupération → Surcompensation
- **Spécificité** : Entraînement adapté aux exigences du sport, transfert des qualités physiques vers la performance

### 1.3 Individualisation
- **Facteurs** : Âge, blessures, calendrier, volume de tatami
- **Surcharge progressive & variation** : Joue sur RPE/RIR, %1RM, ondulation hebdo, blocs de 3–5 sem avec **deload**

## 2) Planification Annuelle (Macrocycle)

### 2.1 Architecture Type (2 pics par an)

| Phase | Durée | Objectif | S&C dominant | Tatami dominant |
|-------|-------|----------|--------------|-----------------|
| **Transition** | 2–4 sem | Décharger / réparer | mobilité, GPP légère | technique légère |
| **Accumulation (Base)** | 4–8 sem | Base aérobie, tissus, hypertrophie fonctionnelle | 6–15 reps, tempo, correctifs | drills techniques, uchi‑komi/positional |
| **Force Max** | 4–6 sem | Recrutement neural | 3–5 reps @85–92% 1RM | randori/sparring modéré |
| **Puissance / Vitesse** | 3–5 sem | Puissance alactique | 1–3 reps @30–70% + plyo | entrées vives, séquences explosives |
| **Spécifique / Rép. efforts** | 2–6 sem | Répéter l'effort spécifique | circuits rép. efforts | sparring dur/tactique |
| **Taper** | 1–3 sem | **Fraîcheur max** | **volume −40/60 %**, intensité maintenue | affûtage, vitesse/tech |

### 2.2 Exemple Annuel (2 pics)
- **Jan–Fév** : Accumulation (GPP/Hypertrophie + base aérobie)
- **Mars–Avr** : Force Max → Puissance (**pic #1 fin avr**) + taper 10–14 j
- **Mai** : Transition courte (deload, mobilité)
- **Juin–Juil** : Accumulation 2 + Force Max
- **Août–mi‑Sept** : Puissance + spécifique (rép. efforts)
- **Fin Sept–Oct** : Compétitions + **taper** (pic #2)
- **Nov** : Transition
- **Déc** : Accumulation légère + correctifs

### 2.3 Cas Usuels
- **2 pics/an** pour amateurs
- **3–4 pics/an** si compétitions fréquentes (blocs plus courts)
- **Saison continue** : micro‑pics, mini‑tapers

## 3) Construire les Blocs (Mésocycles)

### 3.1 Structure des Mésocycles (3 à 5 semaines)
- **Accumulation** : 3–4× S&C/sem, 2–3× aérobie tempo, 2–4× tatami technique
- **Force** : 2–3× lourds, 1–2× puissance légère, tatami 3–5× (moduler la dureté)
- **Puissance/Spécifique** : 2× puissance/plyo, 1–2× force entretien, 3–6× tatami (incl. durs)
- **Deload** toutes 3–5 semaines : −30 à −50 % **volume**; intensité stable

### 3.2 Gestion des Charges d'Entraînement
- **Volume** : Nombre d'heures/séances par semaine
- **Intensité** : Pourcentage de l'effort maximal
- **Densité** : Rapport volume/intensité
- **Fréquence** : Nombre de séances par semaine

## 4) Semaines Types (Microcycles)

### 4.1 Modèles de Périodisation Hebdomadaire

#### Modèle 3+1 (Recommandé)
```
Semaine 1 : 100% (Charge maximale)
Semaine 2 : 100% (Maintien de la charge)
Semaine 3 : 75% (Charge réduite)
Semaine 4 : 50% (Récupération active)
```

#### Modèle 2+1
```
Semaine 1 : 100% (Charge maximale)
Semaine 2 : 100% (Maintien de la charge)
Semaine 3 : 60% (Récupération)
```

#### Modèle 1+1 (Débutants)
```
Semaine 1 : 100% (Charge maximale)
Semaine 2 : 70% (Récupération)
```

### 4.2 Semaines Types Détaillées

#### A) 3 jours S&C + 3–4 tatami (charge moyenne)
- **Lun** : Force bas (hanches) + grip
- **Mar** : Tatami technique + aérobie tempo 20–30'
- **Mer** : Force haut (tirage/poussée) + gainage anti‑rotation
- **Jeu** : Tatami (positional) + alactique 6–10 s
- **Ven** : Puissance + mobilité hanche/cheville
- **Sam** : Randori/sparring (mod‑dur selon phase)
- **Dim** : **Récup active** (Yoga 20') + marche 30–45'

#### B) 2 jours S&C + 4–5 tatami (charge haute tatami)
- **Mar** : Full‑body Force (lourd)
- **Jeu** : Puissance + circuits spécifiques (grip/isométriques)
- Tatami : Lun/Me/Sam (technique + drills), Ven (sparring), Dim (récup)

#### C) 4 jours S&C (pré‑saison) + 3 tatami
- Lundi (Force bas), Mardi (Force haut), Jeudi (Puissance), Samedi (Rép. efforts)
- Tatami : Mar/Jeu (tech), Ven (positional), Dim récup

### 4.3 Rythme de Croisière
```
Lundi : Technique + Conditionnement (80%)
Mardi : Sparring + Renforcement (90%)
Mercredi : Récupération active (50%)
Jeudi : Technique + Tactique (85%)
Vendredi : Sparring intensif (95%)
Samedi : Technique + Mobilité (70%)
Dimanche : Repos complet (0%)
```

## 5) Séances S&C Détaillées

### 5.1 Force — Bas du corps (60–75')
1. **Activation** : breathing + 90/90, Cossack, pont fessier 2×10
2. **Force** : **Trap‑bar DL** 5×3 @85–90% (RPE 8) — repos 3'
3. **Secondaire** : **Split squat bulgare** 3×6/6 (RPE 7)
4. **Chaîne post** : **Hip thrust** 3×8 (RPE 7)
5. **Antirot** : **Pallof** 3×12/12
6. **Grip** : **Farmer carry** serviette/gi 4×30–40 m
7. **Mob fin** : chevilles/ischios 6–8'

### 5.2 Force — Haut du corps (60–75')
1. **Activation** : scapula Y‑T‑W, rotateurs externes 2×15
2. **Poussée** : **Développé couché haltères** 4×6–8 (RPE 7–8)
3. **Tirage** : **Tractions serviette/gi** 5×max‑2 _ou_ tirage poulie 4×8
4. **Superset** : Rowing unilatéral 3×10/10 + Pompes anneaux 3×12
5. **Cou + trap** : shrugs 3×12 + isométrie cou 3×20–30 s
6. **Core** : dead bug / side plank 3×30–45 s

### 5.3 Puissance / Vitesse (45–60')
- Pliométrie : **CMJ** 5×3 ; sauts latéraux 4×4/4
- Balistiques : **KB swing** lourd 6×5 ; med‑ball oblique 6×3/3
- Sprint court (option) : 6–8×10 m (récup complète)
- **Finisher alactique** : 6–10 s très intense / 50–60 s repos ×8–12

### 5.4 Circuits "spécifiques combats" (20–25')
- **Grip & isos** : hang au kimono/serviette 30–45 s → curls serviette 12 → isométrie dorsale 30 s (×3)
- **Rép. efforts** : 20 s row explosif + 20 s fentes sautées + 20 s battle rope / 60–90 s repos (×5–8)
- **Sol** : hip heist 15/15, bear‑crawl 20 m, Turkish get‑up 2/2 (×3)

## 6) Préparation d'une Compétition (8 semaines → Jour J)

### 6.1 Schéma 8 semaines
- **S8–S5 (Build)** : force lourde 2–3×, puissance 1×, 3–5 tatamis (technique prioritaire), circuits rép. efforts **légers**
- **S4–S3 (Intensify)** : force 1–2× (entretien), **puissance 2×**, circuits rép. efforts **plus durs**, sparring spécifique
- **S2–S1 (Taper)** : **−40/60 % de volume**, **intensité maintenue**, séances **courtes & nerveuses**, sommeil + alimentation carré

### 6.2 Détail par Période

#### 8-12 Semaines Avant
- **Objectif** : Développement des qualités spécifiques
- **Volume** : 80-90%
- **Intensité** : 70-80%
- **Contenu** : Technique, conditionnement, sparring

#### 4-6 Semaines Avant
- **Objectif** : Affûtage technique et tactique
- **Volume** : 70-80%
- **Intensité** : 80-90%
- **Contenu** : Simulation de compétition, travail tactique

#### 2-3 Semaines Avant
- **Objectif** : Peak de performance
- **Volume** : 50-60%
- **Intensité** : 90-95%
- **Contenu** : Technique fine, récupération optimisée

#### 1 Semaine Avant
- **Objectif** : Récupération et préparation mentale
- **Volume** : 30-40%
- **Intensité** : 60-70%
- **Contenu** : Technique légère, visualisation, relaxation

### 6.3 Semaine Taper type (J−7 → J−1)
- **Lun** : Puissance légère 35–45' + technique vive 45'
- **Mar** : Récup active (yoga 20') + marche 30'
- **Mer** : Specific speed — séquences 6–10 s / récup complète ×8–12
- **Jeu** : Off ou mobilité + respiration
- **Ven** : Éveil neuromusculaire 20–30' (CMJ, med‑ball, 2–3 sprints 10 m)
- **Sam** : Compétition

## 7) Gestion des Blessures

### 7.1 Phase Aiguë (0-72h)
- **RICE** : Repos, Glace, Compression, Élévation
- **Objectif** : Limiter l'inflammation
- **Activité** : Arrêt complet de l'activité traumatisante

### 7.2 Phase Subaiguë (3-14 jours)
- **Objectif** : Récupération de la mobilité
- **Activité** : Travail de mobilité, renforcement isométrique
- **Volume** : 20-30% de la charge normale

### 7.3 Phase de Réadaptation (2-8 semaines)
- **Objectif** : Retour progressif à l'activité
- **Activité** : Travail technique adapté, renforcement progressif
- **Volume** : 40-70% de la charge normale

### 7.4 Phase de Prévention
- **Objectif** : Éviter les récidives
- **Activité** : Renforcement spécifique, travail de prévention
- **Volume** : 80-100% de la charge normale

## 8) Saison Estivale et Rythme de Croisière

### 8.1 Caractéristiques Estivales
- **Température élevée** : Risque de déshydratation
- **Humidité** : Augmentation de la fatigue
- **Vacances** : Modification des habitudes

### 8.2 Adaptations Estivales
- **Horaires** : Entraînement tôt le matin ou tard le soir
- **Hydratation** : Augmentation de l'apport hydrique
- **Intensité** : Réduction de 10-15% par rapport à l'hiver
- **Récupération** : Augmentation du temps de récupération

### 8.3 Planification Estivale
```
Juin : 80% (Adaptation à la chaleur)
Juillet : 70% (Période de vacances)
Août : 85% (Reprise progressive)
Septembre : 100% (Retour à la charge normale)
```

### 8.4 Variations de Charge
- **Semaine légère** : -20% de volume, -10% d'intensité
- **Semaine normale** : Charge de référence
- **Semaine lourde** : +15% de volume, +5% d'intensité

## 9) Récupération & Prévention

### 9.1 Les Incontournables
- **Sommeil** : 7–9 h, heure fixe, chambre fraîche/obscure
- **Nutrition & hydratation** : protéines ~**1.6–2.2 g/kg/j** (repères), glucides adaptés à la charge, électrolytes si grosses sueurs
- **Récup active** : 20–30' marche/vélo facile + **Yoga/Stretch 20–25'** 2–4×/sem
- **Prévention** : la **force** est l'intervention la plus efficace pour **réduire le risque de blessure** (evidence forte)

### 9.2 Séance Yoga/Stretch 20–25' (détaillée)
1. Respiration **box 4‑4‑4‑4** 2'
2. Flow : **chat‑vache** → **chien tête en bas** (2×30 s) → **fente basse + rotation** (2×30 s/lat)
3. **Ouverture hanches** : pigeon 2×45 s/lat, **90/90** 2×30 s/lat
4. **Chaîne post** : good‑morning bâton 2×12 lent, étirement ischios 2×45 s/lat
5. **Ceinture scapulaire** : thread‑the‑needle 2×30 s/lat, pec doorway 2×30 s
6. Fin : respiration nasale 3–5'

### 9.3 Outils "+" (à bon escient)
- **Bain froid (CWI)** : utile pour diminuer courbatures & améliorer la sensation de récup **à 24 h** après des séances très intenses (ex. 10–15 °C, 10–12 min). Évite‑le juste après les grosses séances d'hypertrophie si l'objectif = adaptation musculaire

## 10) Suivi & Auto‑régulation (sRPE, journal)

### 10.1 sRPE (méthode Foster)
- **sRPE‑Load = RPE séance (0–10) × Durée (min)**
- Exemple : randori 75' à RPE 7 → **525 UA**
- **Charge hebdo** = somme des sRPE
- **Monotony** ≈ moyenne quotidienne / écart‑type (haut = risque)
- **Strain** = charge × monotony

### 10.2 Échelle de Perception de l'Effort (RPE)
- **1-3** : Très facile
- **4-6** : Modéré
- **7-8** : Difficile
- **9-10** : Maximum

### 10.3 Journal Bien‑être (30 sec/j)
- Sommeil (0–3), fatigue (0–3), douleurs (0–3), stress (0–3), motivation (0–3)
- Si 2–3 voyants rouges 3 j d'affilée → **−20/30 % de volume** pendant 3–5 jours

### 10.4 Surveillance de la Fatigue
- **Symptômes** : Fatigue persistante, irritabilité, baisse de performance
- **Marqueurs** : Fréquence cardiaque au repos, qualité du sommeil
- **Tests** : Saut vertical, temps de réaction

### 10.5 Outils de Suivi
- **Journal d'Entraînement** : Volume, Intensité (RPE moyen), Qualité (note subjective 1-10), Récupération
- **Tests de Performance** : Mensuel (saut vertical, temps de réaction), Trimestriel (test de conditionnement spécifique), Annuel (bilan complet)
- **Indicateurs de Santé** : Quotidien (poids, FC repos), Hebdomadaire (sommeil, humeur), Mensuel (bilan sanguin si nécessaire)

## 11) Tests & Repères

### 11.1 Tests Simples
- **Grip** : hang serviette/gi (temps), farmer heavy (distance)
- **Saut** : CMJ (hauteur), sauts latéraux (distance)
- **Sprint court** : 10 m
- **Endurance spécifique** : répétitions d'efforts 15–20 s / 40–60 s repos (volumes tenables)
- **Force** : 3–5RM trap‑bar / back squat / rowing haltère / tractions

### 11.2 Fréquence des Tests
- **Mensuel** : Saut vertical, temps de réaction
- **Trimestriel** : Test de conditionnement spécifique
- **Annuel** : Bilan complet des qualités physiques

## 12) Gestion du Poids (catégories) & Santé

### 12.1 Règles d'Or
- **Évite les "cuts" rapides** (déshydratation, cognition, cardio, blessures)
- Privilégie une perte **progressive** (~**0,5 %/semaine**), planifiée, accompagnée
- **Après pesée** : ré‑hydratation graduelle (eau + électrolytes), glucides faciles, repas testés à l'entraînement (pas d'impro le Jour J)

### 12.2 RED‑S (Relative Energy Deficiency in Sport)
- Surveiller l'**énergie dispo** (apports − dépense)
- Si doutes (fatigue persistante, performances en berne, aménorrhée, blessures récurrentes…), consulter

> _NB_ : Ce guide est pédagogique, ne remplace pas un avis médical/diététique personnalisé.

## 14) Adaptations "Masters" (40+)

### 14.1 Principes
- **Volume ↓**, **intensité conservée** (qualité > quantité)
- **Deload** toutes 3–4 sem
- **Plyo** : volumes modestes, sauts bilatéraux, surfaces sûres
- **Récup** : 2–4 blocs Yoga/Stretch/semi‑off / semaine
- **Force** : 2×/sem full‑body suffisent pour d'excellents gains/maintien

### 16.1 FAQ courte (Q → A)

**Q : C'est quoi un taper efficace ?**
**A :** Baisse le **volume de 41–60 % ~2 semaines**, garde l'**intensité** et la **fréquence** proches de l'habitude. Vise des séances plus **courtes et nerveuses**.

**Q : Combien de sRPE par semaine pour moi (amateur) ?**
**A :** Démarre autour de **2000–3000 UA** totales (tatami + S&C), observe tes voyants (journal). Ajuste ±10–20 % selon feedback.

**Q : Je manque de temps, 2 séances S&C suffisent ?**
**A :** Oui : **Full‑body force** + **Puissance**. Oriente les accessoires vers **grip**, **chaîne post** et **antirotation**.

**Q : Yoga/Stretch : quand ?**
**A :** Après sparring ou les jours légers : **20–25 min** (flow + hanches + scapula + respiration).

**Q : Masters 40+ : principe n°1 ?**
**A :** **Moins de volume, intensité gardée**, deload fréquent, **qualité de sommeil** prioritaire.

**Q : Poids‑catégories : règle d'or ?**
**A :** Pas de cut express. **Perte progressive** (~0,5 %/sem), planifiée et encadrée.

## 17) Glossaire

- **Alactique** : efforts très courts (≤10 s), intensité max, énergie sans lactate
- **Deload** : semaine où l'on baisse surtout le **volume** pour récupérer
- **Monotony / Strain** : indices (Foster) pour estimer le risque de fatigue
- **PPG/PPSp** : préparation physique générale / spécifique
- **RIR/RPE** : reps in reserve / rating of perceived exertion
- **Taper** : période d'**affûtage** avant une compétition (volume ↓, intensité ↔)
- **Tempo** : vitesse contrôlée des répétitions (ex. 3‑1‑1)
- **Macrocycle** : Planification annuelle (10-12 mois)
- **Mésocycle** : Périodes de 3-6 semaines
- **Microcycle** : Semaine d'entraînement
- **Surcompensation** : Phase de charge → Fatigue → Récupération → Surcompensation

## Conclusion

La préparation physique en sports de combat nécessite une approche individualisée et adaptée aux contraintes de chaque pratiquant. La périodisation permet d'optimiser les performances tout en minimisant les risques de blessures et de surentraînement.

Ce guide unifié rassemble tous les aspects essentiels : de la planification annuelle aux séances détaillées, en passant par la gestion des blessures, l'adaptation saisonnière, et les outils de suivi. Il constitue une référence complète pour les athlètes et coachs des sports de combat.
